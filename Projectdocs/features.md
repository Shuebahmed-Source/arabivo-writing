# Features

## Authentication
- User sign up
- User login
- Protected routes for the learning app

## Learning Path
- Structured progression through lessons
- Lessons unlock sequentially
- Clear unit structure (letters, positions, words)

## Writing Practice
- Writing canvas for tracing Arabic letters
- Letter guides shown faintly in the background
- Clear button to reset writing
- Check button to evaluate tracing
- Feedback after evaluation

## Lesson Flow
- Show Arabic letter or word
- Show transliteration
- Allow tracing practice
- Provide evaluation feedback
- Allow retry or move to next lesson

## Progress Tracking
- Track completed lessons
- Save user progress
- Display course progression

## Responsive Design
- Fully responsive interface
- Optimized for phones and tablets
- Desktop compatible
